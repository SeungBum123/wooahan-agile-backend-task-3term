# 👨🏻‍💻 우아한 애자일 Back-End 과제

<br>

## 🖍 문제

- Back-End 서버 구성 및 게시판 CRUD API 구현
  1.  게시판 전체조회
  2.  게시글 상세조회
  3.  게시글 생성
  4.  게시글 수정
  5.  게시글 삭제

<br>
